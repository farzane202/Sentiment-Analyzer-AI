# Sentiment analyzer AI 

This is a small Python project for sentiment analysis of movie reviews.  
It uses a Naive Bayes classifier from scikit-learn to predict if a review is positive or negative.

 Features
- Interactive command-line interface for entering movie reviews
- Predicts whether a review is positive or negative
- Demonstrates basic Machine Learning and Natural Language Processing (NLP)


 How to run
 Install required Python libraries:
   ```bash
   pip install scikit-learn nltk
